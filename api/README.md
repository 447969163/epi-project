#### 初始化
+ 执行`npm install`补全需要的包
+ 在浏览器地址栏输入`http://localhost:3030/接口地址`访问接口

```#### 捐赠一级分类(如医用防护，生活用品等)接口
|接口地址|返回方式|请求方式|请求示例|接口备注|
|:---|:---|:---|:---|:---|
|/api/donation|json|GET|/api/donation|返回捐赠类别名称|
##### 请求参数说明
|名称|必填|类型|说明|
|:---|:---|:---|:---|
|/|/|/|/|
##### 返回参数说明
|名称|类型|说明|
|:---|:---|:---|
|code|int|状态码|
|msg|string|返回码描述|
|dtype|捐赠类别名称|
##### JSON返回示例
```json
{
    "code": 200,
    "msg": "查找成功",
    "result": [
      {
        "dtype":"医疗用品"
      }
    ]
}
```
